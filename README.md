# Stalker
Command line application as a gem to stalk people on twitter. Finds tweets in which the person was mentioned.

## Installation

Checkout the code and run

    $ rake install
         OR  
    $ gem build ./stalker.gemspec
    $ gem install ./stalker-0.0.1.gem

## Usage

    $ stalk <username>
For Example : 
    $ stalk LeoDiCaprio
